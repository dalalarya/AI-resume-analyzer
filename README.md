# AI-resume-analyzer
AI Resume Analyzer using NLP

The AI Resume Analyzer is a Python-based web application that compares a resume with a job description and provides a match score along with improvement suggestions.

It helps users optimize their resumes for ATS (Applicant Tracking Systems).

------------------------------------------------------------------------------

#Features
-  Match score using TF-IDF & Cosine Similarity  
-  Missing keyword detection  
-  Top keyword extraction  
-  Graph visualization of match vs gap  
-  Smart suggestions based on score  

---------------------------------------------------------------------------------

#Tech 
- Python  
- Streamlit  
- Scikit-learn  
- PyPDF2  
- Matplotlib  

------------------------------------------------------------------------------------

# Working 
1. Upload resume (PDF)  
2. Paste job description  
3. System processes text using TF-IDF  
4. Calculates similarity of JD and Resume using cosine similarity  
5. Displays match score and suggestions
